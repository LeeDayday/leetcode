## 핵심 아이디어
- 다각형을 삼각형들로 분할
- 각 삼각형의 점수는 세 꼭짓점 값의 곱
- 전체 점수의 최솟값 구하기
- 어떤 점을 기준으로 삼각형을 만들지에 따라 이후 분할 결과가 달라짐
- 최적 부분 구조(Optimal Substructure)가 존재
- 모든 분할 위치를 시도해야 하므로 DP 사용

## 패턴
- Interval DP
- 구간 분할 DP
- Divide and Conquer + Memoization
- 최솟값 DP

## 문제 재해석
- 구간 [i, j] 를 최소 비용으로 삼각분할한다고 정의
- i 와 j 를 연결하는 변은 고정
- 중간 점 k 를 선택하여 삼각형 하나 생성

```text
(i, k, j)
```

- 생성되는 삼각형 비용

```text
values[i] * values[k] * values[j]
```

- 남은 문제는 두 개의 더 작은 구간

```text
[i, k]
[k, j]
```

## DP 정의

```swift
dp[i][j]
```

- i ~ j 구간을 삼각분할했을 때의 최소 비용

## 점화식

```swift
dp[i][j] = min(
    dp[i][k]
    + dp[k][j]
    + values[i] * values[k] * values[j]
)
```

- 모든 k ∈ (i+1 ... j-1) 시도

## Base Case

```swift
if j - i < 2 {
    return 0
}
```

- 꼭짓점이 2개 이하
- 삼각형 생성 불가
- 비용은 0

## 구간 분할 구조

```text
      dp[i][j]

          k
         / \
        /   \
       /     \

   dp[i][k]  dp[k][j]

+ triangle(i,k,j)
```

## Top-Down Memoization 패턴

```swift
func dfs(_ i: Int, _ j: Int) -> Int {
    if j - i < 2 { return 0 }

    if memo[i][j] != -1 {
        return memo[i][j]
    }

    var ans = Int.max

    for k in (i + 1)..<j {
        ans = min(
            ans,
            dfs(i, k)
            + dfs(k, j)
            + values[i] * values[k] * values[j]
        )
    }

    memo[i][j] = ans
    return ans
}
```

## Bottom-Up 순회 순서

- 작은 구간부터 큰 구간으로 확장

```swift
for length in 3...n {
    for i in 0...(n - length) {
        let j = i + length - 1

        for k in (i + 1)..<j {
            ...

    }
}
```

## 시간복잡도

- 상태 수: O(n²)
- 각 상태마다 k 탐색: O(n)

```text
O(n³)
```

## 공간복잡도

```text
O(n²)
```

## 이 문제에서 중요한 사고

- 현재 선택(k)이 이후 결과에 영향을 줌
- Greedy 불가능
- 모든 분할 위치를 시도해야 함
- "구간을 둘로 나누어 최소 비용을 구한다"는 발상

## 자주 연결되는 패턴

- Matrix Chain Multiplication
- Burst Balloons (312)
- Minimum Cost Tree From Leaf Values (1130)
- Interval DP
- Polygon Triangulation

## 기억할 포인트

- 다각형 분할 → Interval DP
- 구간 [i, j] 의 답을 정의
- 중간 지점 k 를 선택하여 두 구간으로 분할
- 모든 k 를 시도
- Matrix Chain Multiplication 과 거의 동일한 사고방식
- "구간을 나눠서 최적값을 구한다" → Interval DP 신호
