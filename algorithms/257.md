## 핵심 아이디어
- Binary Tree
- root 에서 leaf 까지의 모든 경로 찾기
- DFS 를 통해 트리를 끝까지 탐색
- 현재까지의 경로(path)를 누적하며 내려가기
- leaf node 에 도달하면 결과에 추가

## 패턴
- root 부터 leaf 까지 DFS 순회
- 경로 누적
- leaf node 조건 확인
- 경로 문자열 저장

## DFS 흐름
- 현재 노드 방문
- 현재 경로에 노드 값 추가
- leaf node 라면 결과 저장
- 아니라면 left / right 재귀 탐색

## Leaf Node 판별
- 왼쪽 자식과 오른쪽 자식이 모두 nil 인 경우

```swift
if node.left == nil && node.right == nil
```
