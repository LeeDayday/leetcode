## 핵심 아이디어
- 각 자리 숫자의 합을 반복 계산
- 결과가 한 자리 수가 될 때까지 반복
- 숫자를 문자열로 변환하여 각 자릿수를 순회

## 패턴
- 시뮬레이션(Simulation)
- 문자열 변환
- 자리수 분리
- 반복 계산

## 풀이 흐름
- 숫자를 문자열로 변환
- 각 자릿수를 순회하며 합 계산
- 합계를 다시 문자열로 변환
- 한 자리 수가 될 때까지 반복

## 자리수 분리 패턴

```swift
String(num).map { Int(String($0))! }
```

## 합계 계산 패턴

```swift
digits.reduce(0, +)
```

## 반복 종료 조건

```swift
while strNum.count != 1
```

- 한 자리 수가 되면 종료

## 시간복잡도
- 각 반복마다 현재 자릿수 개수만큼 순회
- O(log n)

## 공간복잡도
- 문자열 및 배열 생성
- O(log n)

## 이 문제에서 중요한 사고
- "각 자리수를 어떻게 분리할까?"
- 문자열 변환을 활용하면 구현이 쉬움
- 반복적으로 동일한 연산을 수행하는 시뮬레이션 문제

## 수학적 최적화 존재
- Digital Root 문제
- 반복 계산 없이 O(1)에 해결 가능

```swift
if num == 0 { return 0 }
return 1 + (num - 1) % 9
```

## 자주 연결되는 패턴
- Digit Sum
- 자리수 분리
- Simulation
- Math (Digital Root)

## 기억할 포인트
- 자리수 문제 → 문자열 변환 또는 %10 / 10 활용
- 반복 계산 → Simulation
- Follow-up 에서 O(1) 수학 풀이(Digital Root) 등장 가능
