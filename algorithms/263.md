## 핵심 아이디어
- Ugly Number는 소인수가 2, 3, 5만 존재하는 수
- 2, 3, 5로 나누어 떨어지는 동안 계속 나눈다
- 모든 2, 3, 5를 제거한 뒤

    - 최종 결과가 1이면
        - 소인수가 2, 3, 5만 존재
        - Ugly Number

    - 최종 결과가 1이 아니면
        - 2, 3, 5 이외의 소인수가 존재
        - Not Ugly Number

## 패턴
- 특정 조건을 만족하는 인수를 반복적으로 제거
- while n % factor == 0 형태로 약수 제거
- 제거가 끝난 뒤 남은 값으로 판별

## 예외 처리
- n <= 0 이면 false
- 1은 Ugly Number
    - 소인수가 존재하지 않지만 문제 정의상 Ugly Number
