## 핵심 아이디어
- 이진 트리 뒤집기
	- 뒤집기 = 왼쪽 자식은 오른쪽 자식으로, 오른쪽 자식은 왼쪽 자식으로
- 재귀로 구현할 수도 있지만, stack / queue를 이용하여 반복문으로도 구현 가능
- 핵심은 모든 노드를 한 번씩 방문하여 left, right를 swap 하는 것
## 원리
- 진행 방향
	- 루트부터 시작하여 모든 노드를 순회
	- DFS(stack), BFS(queue) 모두 가능
- 진행 방법
	- 현재 노드의 왼쪽 자식과 오른쪽 자식을 swap
	- swap 이후의 자식 노드들을 다음 탐색 대상으로 추가

## Stack(DFS) 방식 흐름
1. stack에 root 노드를 넣는다.
2. stack이 빌 때까지 반복
3. 현재 노드를 pop
4. 현재 노드의 left / right를 swap
5. swap 이후의 자식 노드들을 stack에 append

### 예시

0단계 (stack: [A])
```
			A
	B				C
D		E		F		G
```

1단계 (stack: [B, C])
- 이 때 주의할 점: 현재 바뀐 것은 A의 자식 방향 뿐, B의 내부, C 내부, ... 는 아직 그대로
```
			A
	C				B
F		G		D		E
```

2단계 (stack: [B, F, G])
```
			A
	C				B
G		F		D		E
```


3단계 (stack: [B, F])
G는 leaf node

4단계 (stack: [B])
F는 leaf node

5단계 (stack: [E, D])
```
			A
	C				B
G		F		D		E
```


6,7 단계: D, E 는 leaf node

8단계 stack 비어있음. invert binary tree 작업 종료


## 핵심 포인트
- 각 노드는 정확히 한 번 방문됨
- 방문 시 left / right를 swap
- DFS / BFS 모두 가능
- 시간 복잡도: O(n)
	- 모든 노드를 한 번씩 방문
- 공간복잡도
	- DFS(stack): O(h) ~ O(n)
	- BFS(queue): O(w)
