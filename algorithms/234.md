## 핵심 아이디어
- Linked List 가 palindrome 인지 판별
- 앞과 뒤를 동시에 비교해야 함
- Linked List 는 뒤에서 접근이 어려움
- 따라서 중간을 기준으로 뒤쪽을 뒤집어서 비교

## 패턴
- Fast & Slow Pointer
- Linked List Reverse
- 양쪽 비교 (Two Pointer 느낌)
- In-place 조작

## 풀이 흐름
- fast / slow pointer 로 중간 찾기
- 중간 이후의 linked list reverse
- 앞 절반과 뒤 절반 비교
- 하나라도 다르면 false

## 중간 찾기 패턴
- fast 는 2칸
- slow 는 1칸 이동

```swift
while fast != nil && fast?.next != nil {
    slow = slow?.next
    fast = fast?.next?.next
}
```
